This web application is designed to predict potential medical conditions based on input symptoms, utilizing advanced AI techniques.

Developed using the Keras library in Python, powered by TensorFlow.

The deep learning model is trained using the Recurrent Neural Networks (RNNs) algorithm, albeit on a limited dataset.

Model Highlights:

  • Input Layer: Processes symptom data.
  • Dense Layer: A single dense layer with 64 neurons and ReLU activation to extract features.
  • Output Layer: Four neurons for making predictions, with Softmax activation.
  • Loss Function: Categorical Cross-Entropy to measure prediction accuracy.
  • Optimizer: Adam optimizer for efficient parameter adjustment.
  • Accuracy: Achieves around 86% accuracy during training. Note that this may vary with different datasets.

The training dataset is curated from reputable health websites, contributing to the model's understanding of various medical conditions.
